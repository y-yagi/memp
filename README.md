# memp

The `memp` shows the memory usage per command.

## Example

```bash
$ memp
EXE                                                            	RSS
/opt/google/chrome/chrome                                      	5G
/usr/bin/gnome-shell                                           	296.9M
/usr/bin/nautilus                                              	206.7M
/usr/bin/gnome-software                                        	175.9M
/usr/lib/xorg/Xorg                                             	106.2M
```
